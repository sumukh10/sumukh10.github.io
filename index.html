What I've Built

PROJECT_001
Sentinel — Password Breach Auditor
A zero-leak password auditing tool that checks breaches via k-anonymity: only a 5-char SHA-1 prefix is sent to the HaveIBeenPwned API, matching the remainder client-side. Also scores entropy and models offline crack-time against a fast attacker (~10¹¹ guesses/sec).
Vanilla JS Web Crypto API k-Anonymity HaveIBeenPwned Privacy by Design
